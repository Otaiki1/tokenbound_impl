feat(sdk): implement gas estimation for Soroban operations (#216)

Implements comprehensive gas estimation capabilities allowing users to
preview transaction costs before executing contract operations.

Features:
- estimateGas() method in SorobanSdkCore for online gas estimation
- Offline gas estimation in OfflineTransactionBuilder for air-gapped flows
- Detailed cost breakdown: base fee, resource fee, refundable, max fee
- Resource usage metrics: instructions, read/write bytes, entry I/O
- Formatted display helpers for human-readable gas reports
- Utility functions: calculateRecommendedFee, checkResourceWarnings,
  compareGasEstimations

API:
  const gas = await sdk.estimateGas('eventManager', artifact, {
    source: 'GABC...',
    feeBufferMultiplier: 1.3
  });
  console.log(gas.summary);
  // "Estimated gas: 0.0001234 XLM (max: 0.0001481 XLM). Resources: 50000 instructions..."

Files added:
- sdk/src/gasEstimator.ts - Core gas estimation logic
- sdk/GAS_ESTIMATION.md - Documentation and usage examples

Files modified:
- sdk/src/types.ts - GasEstimation, ResourceUsage, GasCostBreakdown types
- sdk/src/core.ts - estimateGas() method with tracing and retry logic
- sdk/src/offline-builder.ts - estimateGasOffline() for offline flows
- sdk/src/index.ts - Export gas estimation utilities

Closes #216
